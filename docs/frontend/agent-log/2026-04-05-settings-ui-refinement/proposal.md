# 제안서: 설정(Settings) UI 고도화 및 사용자 경험 최적화

## 1. 해결 방법 정의
기존의 파편화된 설정 디자인을 통합하고, 네이티브한 감성의 세부 레이아웃을 다듬어 일관된 사용자 경험을 제공합니다.

## 2. 주요 제안 내용
- **툴바 테마 전환기 메뉴화**: 텍스트를 제거하고 아이콘 중심의 `Menu` 컴포넌트로 개편하여 깔끔한 드롭다운 UI 제공.
- **설정 항목 다이어트**: 더 이상 사용하지 않는 `Gemma4`를 삭제하여 메뉴를 단순화함.
- **레이아웃 밸런스 조정**: 언어 피커의 너비를 콤팩트(`140pt`)하게 줄여 좌우 여백을 조절하고, 중복 기능을 제거함.
- **모델 관리 2단 구성**: 
    - 1단: '이름+명령 버튼' 좌우 끝 정렬
    - 2단: '긴 설명+상태 메시지+진행바' 배치
    - 목적: 정보의 계층화와 긴 이름의 모델 지원.
- **라이선스 뷰 프레임 고정**: 최대 너비 제약(`maxWidth: 600`) 및 유연한 프레임 설정을 통해 창 늘어짐(Bug) 해결.
- **사이드바 토글 인터랙션**: 설정 버튼을 한 번 더 누르면 이전 화면(Home)으로 복귀하도록 제어 로직 추가.

## 3. 대안 고려
- **전체 창 스크롤 vs 개별 스크롤**: 설정 창 전체가 아닌 '모델 리스트' 영역만 따로 스크롤되도록 설정하여 설정창 하단 버튼이 항상 위치를 유지할 수 있도록 함.
- **이유**: 사용자가 많은 모델을 보유하더라도 '저장' 버튼에 즉각 접근할 수 있어야 함.

## 4. 기대 효과
- 일관된 시각적 무게감 확보.
- 복잡한 정보를 계층적으로 분류하여 가시성 증대.
- 레이아웃 버그 해결을 통한 시스템 안정성 인상 부여.
