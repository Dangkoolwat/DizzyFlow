# 변경 내역 요약 (워크스루)

## 1. 개요
DizzyFlow의 워크플로우를 전문 수준으로 격상하기 위해 리팩토링된 전체 기능을 요약함.

## 2. 변경된 주요 기능

### 6단계 상태 머신
- `idle`, `ready`, `processing`, `completed`, `failed`, `cancelled` 6개 상태로 확장.
- 각 단계에 최적화된 중앙 워크스페이스 화면 전환.

### 플랫 사이드바
- Home / 문서 리스트 / Settings를 단일 계층으로 단순화하여 가독성 강화.
- 작업 시작 시 문서 자동 생성 및 사이드바 선택 동기화.

### Safe Lock 및 실시간 피드백
- Processing 중 사이드바와 인스펙터를 비활성화하여 안정성 확보.
- 작업 진행률 바와 세그먼트 생성 내용(Mock)의 실시간 누적 스크롤.

### 시각화된 Ready 상태
- 파일 드롭 시 Mock 메타데이터(크기, 시간, 포맷 등) 생성 및 전용 컨텐츠 카드 제공.
- 시작 전 설정을 명확하게 검토할 수 있는 환경 구축.

## 3. 기술적 수정 사항
- **macOS 호환성**: macOS 15+ API인 `onScrollPhaseChange` 대체 및 `.accent` 사용으로 인한 빌드 오류 수정.
- **UTI**: `UniformTypeIdentifiers` 임포트 추가로 파일 드롭 시 타입 안정성 확보.
- **데이터 흐름**: `WorkflowStore`를 단일 출처로 하여 전체 UI 계층에 일관된 상태 전달.
